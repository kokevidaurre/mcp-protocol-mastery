# MCP Protocol Mastery

## Purpose
Technical implementation guide for building with Model Context Protocol (MCP).
Senior engineer audience - assumes familiarity with APIs, protocols, TypeScript/Python.

## Repository Structure
```
/foundations          - Core concepts, architecture, design principles
/01-architecture      - Protocol structure, message format, lifecycle
/02-servers           - Building MCP servers (tools, resources, prompts)
/03-clients           - Building MCP clients, host integration
/04-tools-resources   - Deep dive into tools and resources
/05-transports        - stdio, SSE, custom transports
/06-security          - Auth, validation, sandboxing
/07-patterns          - Real-world patterns and anti-patterns
/labs                 - Working code examples
```

## Content Style
- Implementation-focused (code > theory)
- TypeScript and Python examples
- Production patterns, not toy examples
- Security considerations throughout
